web_karaoke_system
==================

カラオケシステム（WEB)

以前から構想だけ考えていたカラオケシステム（WEB）を此れを機に着手。
構想草案としては以下の通り。

[機能]
●出力機能
・再生リストに蓄えられた曲を順に再生
・曲の再生を一時停止
・（特定時間後に自動再生）
・（再生リストから曲を削除））
●入力機能
・曲の検索
・曲を再生リストに登録
・再生リストから曲を削除
●ETC
・曲の登録
